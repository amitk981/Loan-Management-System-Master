# Sanitised API examples

Public routes and success shapes remain unchanged.

## Generate success

```json
{
  "success": true,
  "data": {
    "loan_document_id": "00000000-0000-0000-0000-000000000001",
    "document_type": "term_sheet",
    "generation_status": "generated",
    "document_id": "00000000-0000-0000-0000-000000000002",
    "file_name": "term-sheet-LO00000801.pdf"
  },
  "meta": {
    "request_id": "req-sanitised",
    "timestamp": "2026-07-14T00:00:00Z",
    "api_version": "v1"
  }
}
```

## Direct/HTTP missing authority

```json
{
  "success": false,
  "error": {
    "code": "FORBIDDEN",
    "message": "You do not have access to this loan application.",
    "details": {},
    "field_errors": {}
  }
}
```

No response or audit payload contains template/generated bytes, merge values, borrower facts,
storage keys, or download descriptors.
